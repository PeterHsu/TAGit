# 綀習1
## 目標
gitLab01
瞭解基本的git flow AVH Edition操作, 常用gitk --all及自訂的git logp觀察每個支線的變化
hotfix基本上都會和develop衝突, 但release要避免衝突,例如當有release的時候, develop不要產生新的點, 不管如何, 與master的合併一定都不會衝突

|線              | 活動| 版控| 操作| 查詢|

|                |       |git flow init|

|develop         |           ||git flow release start v1.0.0.0|
